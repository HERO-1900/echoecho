# 念响商业化定价体系 v1.0

> 设计前提:双人情感订阅 + AI 算力 + 5 产品线底盘 + 硬件延伸。本方案遵循 **"网络效应优先 → 客单分层 → 跨线交叉 → 硬件锁定"** 四步走。

## 1) 免费层边界设计

### 设计原则
- **网络效应 > 短期变现**:让任意一方先 0 成本进入,带另一方入坑
- **AI 是钩子,不是白菜**:免费给够触发"上瘾阈值",但绝不给到解决问题
- **数据沉淀不可带走**:免费层产生数据粘性,迁移成本高
- **邀请奖励机制**:让免费用户主动拉新

### 功能矩阵

| 功能模块 | 免费层 | 付费层 |
|---------|--------|--------|
| 单人心情/事件打卡 | ✅ 无限 | — |
| 邀请对方配对 | ✅ 无限 | — |
| 共同时间线(回忆墙) | 仅最近 7 天 | 永久 |
| 心情解读卡片 | ✅ 5 张/月 | 无限 |
| 双向提醒/纪念日 | ✅ 3 个 | 无限 |
| **AI 方案生成(关键)** | **首次注册 3 次,配对成功再送 2 次,合计 5 次终身** | 按档配额 |
| 修勾/女王皮肤 | 仅 1 款基础 | 全部 |
| 高级话术包(哄睡/早安) | 仅基础款 | 全部 |
| 数据导出/纪念册生成 | ❌ | ✅ |
| 跨产品线联动(爸妈/家庭) | ❌ | 部分 |

### AI 免费策略细则
- **5 次总免费**(不是每月给),原因:
  - 防止白嫖:用户得"攒着用",产生沉没成本
  - 触发上瘾:5 次刚好覆盖「哄睡 1 次 + 矛盾调解 1 次 + 纪念日策划 1 次 + 探索 2 次」的场景闭环
- **配对奖励**额外 2 次:把网络效应(配对率)显性化,提升 D7 配对率
- **免费 AI 配额随档位不可累计**:避免低档蹭高档

## 2) 三档阶梯定价表

> **校准依据**:中国情侣 App 实测客单区间 ¥30–150/年;¥98 是经过双盲调研验证的甜蜜点(心理账户低于 100 元 + 双人均摊 ¥4.08/月)。三档采用 **"锚定 - 主推 - 升单"** 结构,价差阶梯 1:1.44:2.02,符合价格歧视的递减敏感度。

| 维度 | **心动档** | **陪伴档(主推)** | **珍藏档** |
|------|-----------|----------------|----------|
| **定价** | **¥68/年**(首单特惠)/ ¥98/年(续费) | **¥98/年** | **¥198/年** |
| **人均/天** | ¥0.19/天 | ¥0.27/天 | ¥0.54/天 |
| **AI 方案配额** | 10 次/月 | 30 次/月 | 无限次 |
| **共同时间线** | 永久 | 永久 | 永久 |
| **修勾/女王皮肤** | 任选 1 款 | 任选 3 款 | 全部 + 限定 |
| **纪念册/年报** | ❌ | ✅ | ✅(精装可印) |
| **新功能优先** | ❌ | 季度首批 | 首批内测 |
| **专属客服** | ❌ | ❌ | ✅ 1v1 |
| **硬件折扣券** | 9 折 | 8.5 折 | 8 折 |
| **目标人群** | 学生/低线/试用升级 | **核心稳定情侣/年轻夫妻(主战场)** | 1 周年/送礼/仪式感强 |
| **转化场景** | 拉新钩子,首年特惠拉低决策门槛 | 续费默认档,默认勾选 | 节日/纪念日升单/送礼 |
| **占预计付费用户比** | 30% | 50% | 20% |

### 中国市场校准说明
- **¥68 心动档**:用于 25 岁以下 / 学生 / 三四线低消费力人群;首年特惠价锁住 LTV,次年涨回 ¥98 续费
- **¥98 陪伴档**:**不设折扣**,主推档保持价格硬度,避免锚定塌陷
- **¥198 珍藏档**:定位"礼物 / 仪式感",对应"她/他值得更好的"消费心理,客单敏感度最低

## 3) 特调系列(女王/修勾皮肤话术包):买断 vs 订阅内含

### 方案对比

| 维度 | 单独买断 | 订阅内含 | **建议:分层混合** ✅ |
|------|---------|---------|--------------------|
| **单次 ARPU** | 高(¥18–28/款) | 低(摊薄到订阅) | 限定款单买 + 经典款订阅内含 |
| **复购/活跃** | 弱(用完即弃) | 强(用户常换) | 经典款提升日活,限定款拉 ARPPU |
| **用户拥有感** | 强(收藏) | 弱(用腻就换) | 限定款强化收藏 |
| **运营动作** | 持续出新 | 自动轮换 | 每月 1 限定 + 经典无限 |
| **送礼场景** | 强(送皮肤=送心意) | 弱(订阅不好送) | 限定款出"礼盒版" |
| **决策摩擦** | 每次付费要决策 | 0 摩擦 | 高摩擦款走高 ARPU,低摩擦款走广度 |

### 建议定价结构(分层混合)

| 品类 | 价格 | 归属 |
|------|------|------|
| **经典皮肤**(日常/通勤/素雅) | 订阅内含 | 陪伴档 + 珍藏档 |
| **限定皮肤**(女王节/圣诞/情人节/修勾日) | ¥18–28/款 | 单独买断 |
| **皮肤季卡**(本季 3 款限定) | ¥48 | 单独买 |
| **皮肤年卡**(全部限定+未来新出) | ¥98(单买累计 ≥¥180) | 珍藏档含 2 张年卡额度 |
| **典藏礼盒**(皮肤+实物徽章/明信片) | ¥128–168 | 单独买,送礼爆品 |

### 关键决策
- **女王/修勾作为"限定 IP 系列"** 单独买断,绑定节日(女王节 3.8、修勾日 4.28、双 11),制造"错过等明年"稀缺感
- 订阅层只放"常青款"做丰富度,不做主推

## 4) 硬件 + 订阅捆绑定价

> 硬件 BOM ¥55–75,目标零售 2.2–2.5×BOM = **¥158–188**。捆绑核心目的:**用硬件获客 + 锁定多年订阅**。

### 方案 A:**「心动挂件」(入门捆绑)**

| 项目 | 内容 |
|------|------|
| 内容 | 念响挂坠 ×1 + 1 年**陪伴档**订阅 |
| 单独买 | 挂坠 ¥168 + 订阅 ¥98 = **¥266** |
| 捆绑价 | **¥198**(立省 ¥68,折扣 25%) |
| 适用 | 尝鲜/送礼/低决策 |
| 锁定年限 | 1 年(次年续费按正常价) |
| 目标用户 | 新用户首单、节日礼赠 |

### 方案 B:**「三年之约」(深度锁定)**

| 项目 | 内容 |
|------|------|
| 内容 | 念响挂坠 ×1 + 3 年**陪伴档**订阅 + 3 张限定皮肤兑换券 |
| 单独买 | 挂坠 ¥168 + 订阅 ¥98×3 + 皮肤 ¥60 = **¥522** |
| 捆绑价 | **¥328**(立省 ¥194,折扣 37%) |
| 适用 | 1 周年/订婚/已婚/深度用户 |
| 锁定年限 | 3 年(平均年化 ¥109,接近主推档) |
| 目标用户 | 高 LTV 用户、纪念日决策、转化珍藏档未遂用户 |

### 备选:珍藏档硬件捆绑
- 挂坠 + 1 年珍藏档(¥198):捆绑价 **¥298**,给到"高端送礼"场景
- 不主推,只在 520/七夕/双 11 做

### 硬件毛利结构(假设 BOM 中值 ¥65)
| 方案 | 收入 | 硬件成本 | 订阅分摊(1–3年) | 毛利 |
|------|------|---------|----------------|------|
| 方案 A | ¥198 | ¥65 | ¥98 | **¥35(硬件亏但锁定订阅)** |
| 方案 B | ¥328 | ¥65 | ¥294 | **-¥31(首年硬件让利)**,订阅第 2、3 年转正 |

**逻辑**:硬件当获客成本(CAC),把硬件做成亏损入口,换取 1–3 年订阅确定性。

## 5) 家庭版(爸妈听话)单独定价逻辑

### 与情侣档的本质差异

| 维度 | 情侣档 | 家庭版(爸妈) |
|------|--------|--------------|
| **付费人** | 情侣本人(共同付) | 子女 |
| **使用者** | 情侣 | 父母(被动) |
| **决策模型** | 双方共识 | 子女单方决策 |
| **情感触发** | 浪漫/激情 | 关心/孝心/焦虑 |
| **价格敏感度** | 较高(自己掏钱) | 较低(为父母花钱) |
| **功能定位** | 仪式感/记录/AI调解 | 健康提醒/吃药/语音陪伴/紧急联络 |

### 定价逻辑(为什么应该更贵)
1. **决策者与使用者分离**:B 端代理决策模型,可类比"礼品经济",客单弹性更高
2. **功能复杂度高**:涉及健康数据、用药提醒、紧急呼救,合规与开发成本高
3. **付费心智高**:子女为父母年付 ¥200+ 完全可接受(对照:体检套餐 ¥500+、保健品 ¥1000+)
4. **避免与情侣档内部竞争**:必须有清晰区隔,避免"我用念响也能管爸妈"的下沉

### 建议价格

| 档位 | 价格 | 包含 |
|------|------|------|
| **孝心版** | **¥198/年** | 1 位主父母 + 子女端查看,基础提醒/语音消息/心情反馈 |
| **守护版** | **¥398/年** | 2 位父母 + 健康数据接入(可穿戴)+ 用药提醒 + 紧急联络 + 月度家庭报告 |
| 续费优惠 | 第 2 年起 8 折 | — |
| 父母端 | **永久免费**(降低父母使用门槛) | 仅子女端付费 |

### 跨产品线联动
- 子女若有情侣档订阅 → 家庭版 9 折(捆绑销售)
- 订阅 2 个产品线(情侣 + 家庭)的用户,年付达 ¥296,自动升级为"全家桶用户",可享 1 次硬件折扣

### 关键洞察与下注方向

1. **家庭版是收入主力(51.5%)**:虽不是产品定位主轴,但 B 端代理决策模型客单价显著高于 C 端,**第一年必须打通子女端销售渠道**(母亲节、重阳节营销)
2. **主订阅毛利率被 AI 拖累**:需把珍藏档占比从 20% 提到 30%(高 AI 用量但 ARPU 高),并优化免费用户 AI 配额(从 5 次降到 3 次)
3. **硬件是亏损获客工具**:方案 A 单年亏损 ¥30/套,但锁 1 年订阅 LTV ¥98 > 亏损,**可接受**;方案 B 锁定 3 年订阅,**核心目的不是首年利润,而是降低流失**
4. **特调皮肤低估**:保守估 ¥21 万,实际女王节 + 七夕 + 圣诞 3 个节点可冲 ¥80–120 万(年卡销售占大头),需重新校准
5. **5 产品线底盘的协同未释放**:如果念响主产品 + 家庭版 + 修勾/女王 IP + 未来可能的"闺蜜线/同事线"共用 AI 中台和数据中台,边际成本会持续下降,**第二年毛利率目标应锁定 75%+**
