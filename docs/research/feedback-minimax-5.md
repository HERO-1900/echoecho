# MiniMax 用户反馈 #5

> 站在【红队/风险】视角,预测这个产品上线后最可能翻车的 5 个点(社会舆论/情感反噬/隐私/性别争议/被大厂抄),每个给缓解思路。

# 🔴 红队视角:这款产品上线后最可能翻车的 5 个点

> 核心判断:这个产品的浪漫之处,恰恰是它最脆弱的地方。一旦你把「想念」做成了按钮、把「爱」做成了卡片、把「关系」做成了工单,你就在用**工具的逻辑**,解一道根本不是工具能解的题。

---

## 1️⃣ 「进度条」把爱人变成工单 — 任务化爱意

**真实场景:** 周末早上,女生发语音"宝宝,客厅该收一下了"。男生那边弹出一张卡:`客厅收纳 | 进度 0%`。男生去打游戏,下午女生刷 App,看到 `进度 30% | 已办:把遥控器放回原位`。一个本可以一笑而过的小事,在进度条的语言里,被读成了"你在敷衍我"。

更致命的:AI 僚机把"想买洗衣机"变成选购对比页。看上去贴心,实则**把"我们"的生活决策外包给算法**。男生选了一款 3000 块的,女生嫌贵。原本一句"我想要一台洗衣机"可以一起挑的事,变成了"你连我发的方案页都没仔细看"。

**为什么危险:**
- 「三态 + 进度 %」本质上是 **Jira / Slack 工单系统套在爱情上**。
- **念想模式(成熟同居/琐事)首当其冲**:谁洗碗、谁倒垃圾、谁接孩子,全被量化,原本"我懂你"的小默契,被"你 67%"取代。
- 中文互联网对"KPI 式亲密"极敏感。一条截图就能上热搜:`我以为谈恋爱,结果领了个工单 App`。
- AI 僚机一旦做错(推了款 5000 的洗衣机,对方嫌乱花钱),**算法被甩锅给伴侣**,AI 反向变成吵架催化剂。

**缓解思路:**
- **砍掉「进度 %」**,改二元 + 情绪标签(在办 / 在忙 / 有点累),不鼓励量化努力。
- AI 僚机不要替伴侣出方案,改成**「问题清单 + 聊天脚手架」**,把决策还给两个人。
- 加 **「24h 自动溶解」机制**:过夜的鸡毛蒜皮自动消散,避免怨气沉淀成档案。
- 卡片设计禁止出现"超时/逾期"字眼,只允许"已办妥/不需要了/改天再说"。

---

## 2️⃣ 五种人格模式 — 踩遍性别政治的全部雷区

**真实场景:** **拉勾勾(恋爱前中期/甜蜜仪式感)** — 听起来无害,实则成了**「舔狗神器」**。男方每天按 AI 提示给女方发"早安宝宝我好想你",一个月后被女方闺蜜截图发小红书,标题:`恋爱脑能不能用 AI 给女朋友写情书?`,评论区清一色嘲讽,连带品牌被挂。

**女王陛下(双向)** 是更大的雷。营销稿写"双向",但**首屏主图、推送文案、新手引导**,会本能地预设一个"主"一个"从"。要么被女权 KOL 骂"强化性别刻板",要么被男性社群骂"凭什么她就是陛下、我是臣"。

**毒舌修勾里的"小狗"**,在豆瓣 `#服美役` `#puppy play` 相关讨论里,已经是**被高度警惕的亲密关系驯化符号**。一篇深度文就能把整个产品打成"PUA 美化包"。

**为什么危险:**
- 任何含"主从 / 驯化 / 撒娇"的语义,在 2024-2026 的中文互联网都是**舆论燃料**。
- "双向"写在 PR 稿里没人信,要看产品本身是否真的对等。
- 小红书 + 微博 + 豆瓣 三连一波,**还没等到留存数据,品牌就先被挂了**。
- **爸妈听话模式** 同样有坑:子女给父母推送"您已 3 天没给爸妈打电话",**亲情 KOL 会骂"亲情 KPI 化"**。

**缓解思路:**
- 取消预设的主从视角,五模式一律做**「角色 A ⇄ 角色 B」**对称 UI,文案必须过性别中立审核。
- **拉勾勾模式不要 AI 代写情话**,改成"共同记忆小档案"(双方各加一段,凑成完整的一页),避免单方面跪舔。
- 引入**UGC 称呼/表情包社区审核机制**,含驯化/物化倾向的标签直接限流。
- 提前写好**三种主从争议下的舆情回应剧本**,别等媒体来问才想话术。

## 3️⃣ 分手后,这 App 就是一颗核弹 — 亲密关系取证化

**真实场景:** 两人同居 2 年,App 里沉淀了 800+ 工单、1200+ 语音、5 种人格下的聊天记录。男生提分手。女生:
- 导出所有"未办"工单 → 朋友圈 `他答应我的事,两年没办完`。
- 拉出某次他没及时回复的语音 → 剪辑成"他根本不爱我"的证据小视频。
- 硬件挂坠退回男生,男生在闲鱼挂 `前女友赠,故事可聊`,被女方新对象刷到。

更阴的:若涉及家暴、出轨、财产纠纷,**全程带时间戳的语音 + 任务卡 + 进度**,就是现成的电子证据。律师会当成神器,媒体会当成"亲密关系监控"反面教材。

**为什么危险:**
- 用户不为证据付费,但**律师、记者、前任,会替你的用户用**。
- 一次爆款负面("我用 XX App 收集到前男友出轨证据")上热搜,品牌连带遭殃。
- 长期留存的内容资产,**90 天后大概率变成法律负债**。
- iOS 隐私新政 / 网信办数据合规,直接卡你"为何要永久保存伴侣语音"。

**缓解思路:**
- 关系状态可一键切到「已分手」,双方独立确认后**72 小时内双向抹除**所有原始数据(只留极简摘要供反思)。
- 原始语音默认**边录边转写即焚**,服务器只存文本,不存音频。
- 任何"取证式导出"必须双方人脸识别 + 短信验证,留痕但阻断单方面滥用。
- 法务明示"不构成法律证据",并与公证处合作提供"情感存档"中转服务,**把证据需求引出你的产品**。

## 4️⃣ 硬件挂坠 — 从浪漫道具到监控 / 家暴入口

**真实场景:** 想象凌晨 2 点,挂坠在床头柜震了 6 次。是你妈按的(爸妈听话模式),她正在医院挂急诊,想让你"感觉到她"。**这不是浪漫,是焦虑放大器**。

更糟的:控制型伴侣,把挂坠当成**单向情绪宣泄口**。"我想你了"按键按到对方没法开会、没法睡觉、没法接电话。接收方不能屏蔽、不能静音、不能拒收(因为产品定义"这是爱的表达")。

**真正的雷,是这玩意儿是实物**:分手后,寄回 = 心理创伤的二次播放;不寄回 = 物理上被纠缠;闲鱼转卖 = 公开羞辱。

**为什么危险:**
- 穿戴设备 + 双向通讯 + 默认"必须接收",**几乎踩遍了"新型家暴工具"的所有设计要素**。
- 一旦媒体做一篇 `亲密可穿戴设备如何成为家暴工具` 专题,你就是反面教材第一名。
- **老年人误触**(爸妈听话模式大字版) — 老人按一下,子女开会中手机疯狂震 20 次。
- 挂坠上的麦克风 / 蓝牙协议被逆向,变成**录音器**或**定位信标**,公关灾难。

**缓解思路:**
- 接收方拥有**绝对勿扰权**:勿扰时段、白名单、震动强度、单日次数上限,**全由接收方控制**。
- 7 天无双向互动,挂坠**自动进入"思念休眠"**,硬件灯光变暗,提示"也许该打个电话"。
- 分手时挂坠走**"情感化退役"流程**:App 内双方各说一段话,远程写入挂坠,挂坠必须物理按一次才能"放行"丢弃,仪式化地闭合关系。
- 硬件**只走近场通讯(NFC / 超宽带),无 GPS、无麦克风、无录音**,从源头切断"位置追踪"与"监听"的想象。

## 5️⃣ 微信 / iOS / 小米三方夹击,你的护城河只剩 6 个月

**真实场景:** 把你的核心交付拆开:
- **微信**:待办、共享相册、拍一拍、状态、视频号、小程序,任选三项就能拼出来,且**自带关系链**。
- **iOS 17/18**:共享相簿、同播共享、信息协作、健康分享,Apple 把"亲密功能"做成了系统级。
- **小米 / 华为 / OPPO**:手环 + 智能家庭 + 健康 App,塞个"情侣模式"是季度 OKR。

你的护城河?**情感设计 + 5 种人格 prompt + 一个挂坠 SKU**。大厂两周抄完功能,半年做得比你好。

商业模式上,情侣 App 的付费意愿**天然就是双倍阻力**(双方都要付,等于单价 ÷2),AI 僚机的 token 成本又高,**LTV / CAC 算不过来**。等不到 IPO,数据先凉。

**为什么危险:**
- 工具型社交产品,在中国**几乎没跑通过**(探探 / Soul 都被微信关系链吸走过)。
- "情侣"场景更糟:用户**只在热恋期 3-6 个月高频用**,之后必然跌。
- 大厂不抄你的 App,抄你的**功能塞进微信 / QQ / 手环**就行,你连谈判桌都上不了。
- 硬件挂坠的供应链、售后、合规(无线电 / 电池),对小厂是**吞金黑洞**。

**缓解思路:**
- **别做平台,做"关系咨询 / 情感 SaaS"**:AI 僚机升级为情侣关系教练,按月订阅,做内容 + 服务,工具只是入口。
- 硬件做**首饰化、奢侈品化**(施华洛世奇联名、银饰定制、IP 联名),单价拉到 800-1500,**大厂懒得做这个 SKU**。
- 把五种人格模式做成**IP + 联名**:拉勾勾出表情包、女王陛下出播客、修勾出毛绒周边 — 用内容养品牌。
- **早做"被并购剧本"**:被字节 / 腾讯 / 小米收,而不是死磕独立 IPO。工具层最终会回到大厂怀里,内容层和 IP 才值得长期持有。

---

## 一句话总结

> 这个产品最大的翻车点,不是功能不够好,而是**它对"亲密"的理解,太工程师了**。把"想念"做成按钮的那一刻,你就输了一半。剩下的那一半,输给了微信、舆论,以及——**人类本来就不需要 App 提醒自己爱一个人**。
