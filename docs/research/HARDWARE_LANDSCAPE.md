### 1.2 逐个短评(目标人群 / 核心场景 / 短板)

**Plaud Note / NotePin**
- 人群:海外知识工作者、销售、医生、咨询;场景:会议纪要、通话录音(Note 卡片贴手机背面借结构传导录通话)。
- 短板:订阅制被用户诟病(转写分钟数墙);录完 → 摘要的产出是"文档",不是"行动";双人协作为零——录音属于"我",没有"给你"。

**Anker soundcore Work(录音豆)**
- 人群:会议/网课人群;场景:开会挂胸前,双击标记重点。硬币尺寸 + 充电盒是它对 Plaud 的差异点。
- 短板:8h 单次续航靠充电盒续命;强绑订阅;仍是"单人效率工具",输出是纪要和待办列表(给自己)。

**钉钉 DingTalk A1**
- 人群:中国职场,钉钉存量用户;场景:会议转写 + 直接进钉钉工作流(闭环是它最强的一点:录音 → 转写 → 钉钉待办/日程)。
- 短板:强绑钉钉生态,离开办公语境毫无情感属性;6 麦 8m 拾音的堆料方向和"两个人的悄悄话"完全相反。

**科大讯飞录音笔系列**
- 人群:记者、律师、学生、专业速记;场景:长时间高保真采访/庭审/课堂,机内离线转写是护城河。
- 短板:¥999–3000+ 客单、带屏带系统的"专业设备"路线,重、贵、学习成本高;完全无社交/亲密属性。

**Limitless Pendant**
- 人群:量化自我(quantified self)极客;场景:all-day always-on 记录人生,做"记忆假体"。
- 短板:always-on 的隐私争议最大(录到别人);100h 续航是为"全天录"服务的,佩戴者要向周围人解释"我不是在偷录你";订阅 $19/月。

### 1.3 为什么它们都偏商务会议?

四个结构性原因(这也是我们的空档所在):

1. **付费能力与订阅逻辑**:转写/摘要是持续的云成本,只有"效率有价"的 B 端/职场人群愿意付月费($15.99–19/月)。消费/情感场景撑不起订阅,所以没人做。
2. **会议是"长录音"刚需**:30 分钟以上的音频才凸显 AI 摘要价值,厂商自然向长续航、多麦克风、远场拾音堆料(钉钉 6 麦 8m、讯飞 2+6 麦)。10 秒钟的一句话留言在他们的产品逻辑里"不值得做"。
3. **ROI 故事好讲**:对老板说"省下写纪要的一小时"容易成交;"让 TA 记得你说过的话"无法进 PPT。
4. **路径依赖**:讯飞从速记起家、钉钉从办公协同起家、Plaud 从通话录音起家——基因都是"记录已发生的信息",没有一家的原点是"向另一个人传递"。

结论:全行业都在做「录音 → 转写 → **给自己**的纪要」,没有人做「录音 → 转写 → **到 TA 的清单**」。这是关系型(dyadic)语音硬件的空白。

## 2. 我们的差异化定位:「情侣间的轻量传话信物」

**一句话:按一下,话就到 TA 清单里。**

### 2.1 单点是什么(瑞士军刀式单点)

只做一条流水线:**录音(≤60 秒)→ 云端转写 → 生成一张任务/念想卡 → 落进 TA 的清单 → TA 手机轻推送**。
明确不做:会议纪要、长录音、说话人分离、多语言翻译、always-on、屏幕。每一项"不做"都同时省下 BOM、续航、隐私风险和解释成本。

"瑞士军刀式单点"的意思:竞品是瑞士军刀(录音+转写+摘要+翻译+模板+助理角色,钉钉 A1 内置 30 种场景模板),我们是那把只有一个刀刃、但刀刃对准一个此前不存在的动作的刀——**异步的、有实体仪式感的"我跟你说个事"**。

### 2.2 为什么轻便 + 简洁 + 低成本 + 情感化能错位竞争

- **技术门槛反着用**:竞品的护城河(麦克风阵列、远场拾音、长续航、机内转写)在 0.5 米对嘴说话的场景全部失效——单麦 + 300mAh 就够。我们把省下的 BOM 花在**材质、造型、开箱仪式感**上,这是效率厂商的组织能力做不出来的。
- **订阅逻辑反着用**:一句话 10 秒音频的转写成本约为会议场景的 1/100 量级(推理:按分钟计费的 ASR,10s vs 30min),免费额度可以真·无限感,砍掉用户最恨的"分钟数墙"。
- **购买理由不同赛道**:竞品是"买给自己的生产力",我们是"买给关系的信物"——对标的不是录音笔,是情侣手环/异地恋礼物(礼赠市场定价宽容,双件装天然合理)。
- **防守性**:大厂看不上(客单低、无 B 端)、录音笔厂做不来(品牌基因是效率不是亲密)。风险点也要说清:该品类天花板低于 B 端 SaaS,靠系列化配件和双件装复购撑量,不靠订阅 ARR。

## 4. 技术路径

### 4.1 BLE 直连手机 App 代传 vs Wi-Fi 直连云端

| 维度 | BLE + App 代传 | Wi-Fi 直连云 |
|---|---|---|
| 功耗 | 极低,300mAh 可待机数周(参照 Omi nRF52 方案) | Wi-Fi 收发功耗高一个量级,需大电池 |
| BOM | nRF52/ESP32-C3 级,便宜 | 需更强 SoC + 天线调试 |
| 配网 | 无(App 扫码即配对) | 配网是消费硬件第一大客诉源 |
| 依赖 | 手机需在 ~10m 内(情侣场景:手机永远在身边,可接受;离线时本地缓存,回连补传) | 可完全脱离手机 |
| 隐私 | 音频经用户自己手机上传,链路可解释 | 设备直传云,信任成本高 |

**结论:MVP 走 BLE + App 代传**(挂链/背贴/项链全部适用);Wi-Fi 直连只留给三期桌面摆件(插电或大电池,且"家里的 Wi-Fi"心智自然)。参照:Omi 开源方案即 BLE + Opus 编码流式传手机(https://github.com/BasedHardware/omi ),协议可直接借鉴。

### 4.2 麦克风阵列必要性:**不必要**

阵列的价值在远场(3–8m)拾音与波束成形(钉钉 6 麦 8m、Plaud Pro 5m)。我们的声源在 0.2–0.5m 的嘴边,单颗 MEMS 麦 + 软件端 NS(RNNoise 级)即可;最多加第二颗麦做简单降噪(推理:参照 Anker 录音豆青春场景与 DJI Mic Mini 单咪成功先例)。省下的不只是 BOM,还有功耗与结构空间。

### 4.3 隐私与合规要点

1. **实体录音键,按住才录(PTT)**:从交互上排除 always-on,这是与 Limitless 划清界限的核心设计——我们不是"记录仪",是"传话筒"。松手即停,最长 60s 硬上限。
2. **录音灯示**:录音时 LED 常亮,固件级绑定(不可软件关闭)。参照各国对录音设备显性提示的普遍要求;国内《个人信息保护法》处理个人信息需告知同意,产品叙事上"给自己人留言"而非采集第三方,风险面小但灯示仍是底线。
3. **不做隐蔽外观**:不伪装成纽扣/饰钉类隐蔽形态——国内对"伪装式窃听窃照器材"有刑事红线(推理/提示:具体以《刑法》284 条及禁止生产销售间谍器材相关规定为准,量产前过一次法务)。
4. **数据链路**:音频转写后默认删除原音频(可选保留"声音原声"作为情感功能,显式开关);端到端仅两人可见;导出/删除自由。
5. **认证成本预留**:BLE 产品国内 SRRC + 3C,出海 FCC/CE,预算 ¥5–15 万量级(估算),众筹定价要摊进去。

## 5. MVP 建议

- **形态**:**「念想磁吸」MagSafe 磁吸挂坠二合一**——圆形/椭圆小坠,背面 MagSafe 磁环吸手机背面,顶部留孔穿挂绳变手机挂链。一个模具吃下形态 1+2,把"吸在手机上"当默认态(解决"忘带"),"挂起来"当情感态。
- **规格**:BLE 5.x 单麦 PTT,按住说话 ≤60s,RGB 呼吸灯(对方收到/给你留言两种灯语),磁吸触点充电,重量目标 ≤15g。
- **目标 BOM**:¥55–75(估算,含结构与包装,不含认证摊销);双件装 BOM ¥110–150。
- **众筹定价**:**双件装 ¥299–349(早鸟 ¥249)**,单件补充装 ¥169。参照锚点:Omi $69.99、Limitless 早鸟 $99、Plaud/Anker $159——我们必须显著低于"效率录音机"的 $159 心智线,双件装对标"情侣礼物 ¥300 档"。无订阅:短语音转写成本低,免费管够(成本推理见 2.2)。
- **过渡策略(硬件出货前就上线)**:
  1. App 先行:手机端长按录音 → 转写 → 落 TA 清单,跑通全链路与留存数据;
  2. **现成设备适配**:iPhone Action Button / 快捷指令、Apple Watch 复杂功能、AirPods 长按唤起"给 TA 留话";DJI Mic Mini 等作为"发烧友外设"适配(磁吸佩戴 10g、TX 11.5h,官方参数 https://www.dji.com/mic-mini/specs ),验证"实体按键→留话"的行为是否真的高频;
  3. 用 ¥10 级蓝牙拍照遥控按钮(BLE HID)做百人级灰度实验件,一周内可发货,先验证"桌上有个实体键"与"手机里一个按钮"的使用频次差,再决定是否 all-in 自有硬件。
