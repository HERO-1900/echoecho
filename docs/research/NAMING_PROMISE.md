## 0. 先回答三个特别探索问题(结论前置)

### 0.1 「拉勾勾」能否升级为主品牌?——**不建议,风险被严重低估**

三条独立证据:

1. **「拉勾」是知名在先商标。** 拉勾网(北京拉勾网络技术有限公司,2013 上线,2017 起前程无忧控股 60%)是互联网招聘头部品牌,旗下含"拉勾网 / 拉勾招聘 / 拉勾 App / 拉勾云人事"。其第 9 类(可下载软件)、第 42 类(软件开发)、第 35 类几乎必然已布满。**「拉勾勾」对「拉勾」构成典型的"呼叫近似 + 含义近似"**,叠字不构成显著区别——这是商标近似审查里最容易被驳回的一类。
2. **ASO 天然被压制。** App Store 中国区搜「拉勾勾」,返回结果 100% 是招聘类(猎聘、前程无忧、赶集招聘…),没有一条相关。用户搜索这个词的意图已经被招聘品类彻底占据,新品牌在语义上是"寄生"而非"独占"。
3. **英文名 Pinky Promise 也已拥挤。** App Store 美区实测同名/近名 App 至少 4 个,且**其中两个就是承诺追踪品类**:`Pinky Promise App`(Make Directory Developers, LLC)、`Pinky Promise Me`(Mateo Cardona);另有 `Pinky Promise Health` 的运营主体直接叫 **Pinky Promise Private Limited**——这是一家以此为**公司名**的实体。中英双端同时撞车。

> **判断:「拉勾勾」是个好的"概念/母题",但是个差的"注册资产"。**
> 正确姿势是——**把"拉勾"降级为品牌故事和交互动作(拉一个勾 / 勾成了),把品牌名让给一个能独占的词。** 落地页的整套视觉语言(小指、印章、说到做到)可以 100% 保留,只换名。

### 0.2 「打勾 ✓」的品牌化潜力——**这是本疆域最大的金矿**

✓ 这个符号一符三关,是极其罕见的语义密度:

| 层 | 含义 | 对应产品 |
|---|---|---|
| 动作层 | **拉勾**(小指勾在一起) | 品牌母题、挂坠造型 |
| 状态层 | **打勾 = 完成 / 办妥** | 三态终点、印章 |
| 通信层 | **✓ = 已送达,✓✓ = 已读**(WhatsApp/Telegram 全球心智) | 三态起点「收到」 |

而且它天然是**数数的**:一个勾 = 收到,两个勾 = 在办,三个勾 = 办妥。**产品的三态,本身就是勾的个数。** 这意味着:logo 是它、进度条是它、印章是它、挂坠是它、Push 图标是它、甚至 App 名就是它。这是"名字 = 符号 = 机制"三位一体,比任何一个抽象好词都值钱。

**基于此,本疆域最强的一批候选(双勾 / 三勾 / 勾妥)都建立在这个符号上,而不是建立在"拉勾"这个已被占用的词上。**

### 候选 01 · 双勾

**名字**|中文:**双勾** · 读音:shuāng gōu · 英文:**Twotick**(造词)/ 备选 `Two Ticks`

**含义与概念**
一个字面上什么都对的名字。**✓✓** ——这两个勾,是全世界年轻人已经被 WhatsApp / Telegram 训练了十年的符号:一个勾是"送达",两个勾是"**她知道你看到了**"。而这个产品要解决的核心痛点,恰恰是落地页上那句被访者原话:「**他明明回复了'好的',但那两个字根本没进他脑子里**」——已读不回是最伤的沉默,双勾就是沉默的解药。
第二关:**两只小指勾在一起,本来就是两个勾。** 拉勾的"勾"和打勾的"勾",在这里合体。
第三关:**双 = 两个人。** 这是一个双向产品,不是待办清单——名字里就写着"这事得两个人才成立"。
故事讲法:「一个勾是我听见了,两个勾是我做到了。」

**视觉特征**
- Logo:两个 ✓ 交叠,但**第二个勾的起笔从第一个勾的收笔处长出来**——形态上是两只小指钩住,阅读上是 ✓✓。极简、可做单色、可缩到 16px 还认得出。
- 主色:**朱砂红 #C73E3A**(承接现有印章色)+ 奶油底 #FFF8F0;第二个勾用 pink #F4A7B9,做"她的勾 / 他的勾"的色彩分工。
- Motif 系统(天然成体系):**勾的个数 = 状态**。灰色空勾(未读)→ 单红勾(收到)→ 半勾+进度(在办)→ 双勾+印章(办妥)。挂坠就是一枚 ✓✓ 金属片,两笔一体成型,做项链/钥匙扣毫无违和。

**表达思路**
- 语调:轻、准、不肉麻。像一条已读回执,而不是一首情诗。
- Slogan:「**一个勾是听见,两个勾是做到。**」/ 副:「已读不算数,勾上才算数。」
- 三态话术:**收到 = 勾上了** ｜ 在办 = 勾着呢 ｜ 办妥 = **双勾** (盖章文案:"✓✓ 办妥")
- 五线延展:① 说一句 → 她那头亮起第一个勾 ② 在办 = 第二个勾正在画(勾的描边动画就是进度条,进度即笔画) ③ 办妥 = 第二个勾闭合并盖红章 ④ AI 僚机 = "帮你把第二个勾画完" ⑤ 挂坠 = ✓✓ 吊坠,他画完第二个勾,她的坠子震一下。

**占用/商标风险**|**低—中**
- App Store 中国区搜「双勾」:**无同名 App**(返回全是扑克记牌器,零相关)。中文侧干净。
- 美区 `DoubleTick`:已有 `DoubleTick`(Deepak Bhagchandani,WhatsApp 营销工具)与 `Double Tick LifeStyle`——**所以英文不用 DoubleTick,改用造词 `Twotick`**(实测美区无同名返回)。
- 风险点:「双勾」在书法/绘画里是既有术语(双钩填墨),第 16 类可能有占用,但与 9/42 类不冲突。

**锐评**|**本疆域综合第一。** 唯一一个"名字=符号=产品机制"完全同构的候选,且中英双端都能拿下。缺点是它有点冷——它赢在聪明,不赢在动人,情感浓度不如"不许变"。

### 候选 02 · 不许变

**名字**|中文:**不许变** · 读音:bù xǔ biàn · 英文:**No Take-Backs**

**含义与概念**
「拉钩上吊,一百年不许变。」——**不取前半句(已被拉勾网占死),取后半句。** 这三个字是整句童谣里情绪最重的落点:不是"我们约好了",是"**你不许赖**"。它带着小孩子式的、不讲道理的、可爱的强硬。
英文侧是罕见的完美对位:英美小孩拉勾时喊的是「**No take-backs!**」(不许反悔)——**这不是翻译,这是同一个仪式在另一个语言里的原句。** 一个中文品牌能在英文里找到"原生对应句"而不是"翻译",是可遇不可求的。
故事讲法:「所有认真的承诺,都是从一句不讲理的'不许变'开始的。」

**视觉特征**
- Logo:一枚**红色小指印章**,印文就是"不许变"三个字(或一个变体:勾住的小指,压在一个印泥红方块上)。整个品牌可以只用**印章**这一个器物。
- 主色:**印泥红 #C73E3A** 独大,配纸白/奶油。这个名字气质偏"手写、纸质、盖章",极适合做**信物感**,和硬件挂坠的调性天然一致(挂坠 = 一枚可以随身带的小印章)。
- Motif:印章、指纹、朱砂、按下去的那一下。可延展出「盖章音效 / 震动反馈」这种极强的仪式性交互。

**表达思路**
- 语调:嘴硬心软。有一点撒娇,有一点较真。是这批候选里**唯一有人格**的名字。
- Slogan:「**说过的话,不许变。**」/ 副:「小时候的规矩,现在也算数。」
- 三态话术:收到 = **算数** ｜ 在办 = 正在变成真的 ｜ 办妥 = **没变**(印章文案:"一百年不许变")
- 五线延展:① 她说一句 → 变成一条"不许变"的约定 ② 进度 = 印章从空心一点点填实 ③ 办妥 = 印章按下,可回看的"信物册"(所有盖过的章按时间排成一墙,这是留存与情感复购的核心) ④ AI 僚机 = "别让它变" ⑤ 挂坠 = 印章造型吊坠,可实际蘸印泥盖在纸上的**真章**(这是能卖钱的物件)。

**占用/商标风险**|**低(本批最低)**
- App Store 中国区搜「不许变」:**零结果**(`resultCount = 0`)。搜「一百年不许变」:**零结果**。
- 美区 `NoTakeBacks`:零同名返回。
- 三字口语短句作商标显著性足够(参照"钉钉""得到""刻画"),且不描述功能,不易被以"缺乏显著性"驳回。
- 风险点:它是一句**通用俗语的一部分**,别人也能用这句话做营销(不能阻止竞品说"一百年不许变"),但能阻止别人注册同名。

**锐评**|**情感天花板最高,可注册性最好,但它不像一个 App 的名字。** 用户在应用商店里看到"不许变"三个字,第一反应是懵,第二反应才是笑。这是一个需要"配一句话才能懂"的名字——好在这一句话极其好讲。**赌它,就是赌品牌力大于搜索便利。**

### 候选 03 · 勾妥

**名字**|中文:**勾妥** · 读音:gōu tuǒ · 英文:**Gotto**(造词,谐音 "gotta" = 我得去办)

**含义与概念**
纯造词,不是任何现成的中文词——**这是本批可注册性最强的结构。**
「勾」= 拉勾(承诺发生)+ 打勾(承诺完成);「妥」= 办妥(现有三态的终点原话)。**两个字压缩了产品的全流程:勾上了,就得妥。**
英文 `Gotto` 谐音英语口语 `gotta`(I gotta do it = 我得去办这事)——**男生视角的第一反应,恰恰就是产品要触发的行为。** 一个中文造词能在英文里落成一句口语动词,很罕见。
故事讲法:「勾了,就得妥。」——五个字,产品说完了。

**视觉特征**
- Logo:一个 ✓ 的收笔处**折成一枚小旗/一颗钉子**,读作"勾住并钉牢"。或更极简:把「妥」字的宝盖头 + 女字底,抽象成"一个勾罩着一个人"。
- 主色:靛蓝 #2F4F6F(理性、可靠、男性侧不排斥)+ 朱砂红点缀。**刻意不走全粉——这是这批里唯一一个可以做成"中性/男生也愿意开"的品牌。**(现有 GTM 有个隐患:粉红+小指+印章,会把办事方那一侧劝退。)
- Motif:勾 + 钉 + 落槌。声音设计:"咔哒"一声(钉住的声音)。

**表达思路**
- 语调:干净、利落、略带工程师式的爽快。像 `merged`。
- Slogan:「**勾了,就得妥。**」/ 副(男生视角):「她说的每一句,我这儿都有个去处。」
- 三态话术:收到 = **勾上** ｜ 在办 = 妥办中 ｜ 办妥 = **妥了**
- 五线延展:① 一句话 → 一个勾 ② 进度% = "离妥还有 35%" ③ 办妥 = "妥了"+交待 ④ AI 僚机 = **"帮你妥"**(这是最好的僚机动词) ⑤ 挂坠 = 勾+钉造型,金属感,男生戴得出去。

**占用/商标风险**|**低**
- App Store 中国区搜「勾妥」:**无同名**(返回全是相亲交友类,零相关)。
- 美区 `Gotto`:无同名 App;近似 `Gotta Apps`(瑜伽 App 开发商)存在,类别相邻(9 类),需做检索——**这是唯一的中风险点**。
- 造词,显著性强,9/42 类拿证概率高。

**锐评**|**最像一个"能活下来"的产品名。** 不炫技、不肉麻、好念好打好记,而且是这批里唯一认真考虑了**男性用户不反感**的名字。缺点:故事性弱于"不许变",第一眼看不出是情侣产品——但这可能恰恰是优点。

### 候选 04 · 三勾

**名字**|中文:**三勾** · 读音:sān gōu · 英文:**Trikk**(造词)

**含义与概念**
**产品的三态,就是三个勾。** 收到打第一个勾,在办打第二个勾,办妥打第三个勾——**集齐三个勾,这件事才算过去。** 名字直接是产品的核心机制,不需要额外解释,用户用一次就懂。
副线:古汉语"三"表"再三、郑重"(三顾、三缄其口),「三勾」自带"这不是随口一说"的分量。

**视觉特征**
- Logo:三个 ✓ 呈阶梯上升排列(第三个最大且被红圈圈住 = 印章),本身就是一个**进度指示器**。
- 主色:朱砂红 + 三档灰阶(未勾/半勾/已勾)。
- Motif:三个勾的"集齐"动画,可做成极上瘾的微交互(第三个勾落下时印章"啪"地盖上)。

**表达思路**
- 语调:清晰、有节奏、像口令。
- Slogan:「**三个勾,一件事。**」
- 三态话术:直接就是 一勾(收到)/ 二勾(在办)/ 三勾(办妥)——**话术即品牌,零成本。**
- 五线延展:①→⑤ 全部围绕"集勾"。挂坠可做三格,亮到第三格 = 办妥。

**占用/商标风险**|**中**
- App Store 中国区已有「**三勾商城**」「**三勾门店**」(Wuhan Jiujiujia Network Technology Co., Ltd.)——**同名已被用于电商 SaaS,大概率已注册 9/42 类。** 这是硬伤。
- 英文 `Trikk` 干净,但中文侧要打近似战。

**锐评**|**机制最完美,可注册性最脏。** 如果「三勾商城」的商标只覆盖 35 类(电商),还有一线生机;若覆盖 9/42,直接出局。**要不要它,取决于一次真实的商标检索——但从产品逻辑上说,它是这批里最"自解释"的名字。**

### 候选 05 · 指印

**名字**|中文:**指印** · 读音:zhǐ yìn · 英文:**Pinkprint**(造词:pinky + fingerprint + blueprint)

**含义与概念**
三关叠加,是本批语义密度最高的一个:
1. **指印 = 手指按下的印**——古人签契约不识字就按手印,**"按了手印"= 中国人对"不可反悔"最原始的理解。**
2. **指 = 小指**——按下的是那根拉过勾的手指。
3. 英文 `Pinkprint` = **pinky(小指)+ blueprint(方案图)**——而 AI 僚机干的活,就是把她的一句话变成一张**方案蓝图**。一个词把"承诺"和"方案"两个产品支柱缝在一起。
故事讲法:「不用签字,按个指印就作数。」

**视觉特征**
- Logo:一枚**指纹**,但指纹的纹路末端**收成一个 ✓**。(极强、极独占、一眼记住,且天生适合做 App 图标——圆形指纹在圆角矩形里非常稳。)
- 主色:印泥红 + 米纸白。
- Motif:指纹、按压、留痕。交互:**长按屏幕上的指纹圈来"确认收到"**——一个物理动作等于一次契约,比点按钮重得多。硬件挂坠上就是一块可按的指纹感应面。

**表达思路**
- 语调:郑重、有分量、有旧时代的手工感。反算法、反效率、亲手做。
- Slogan:「**按个指印,这事就作数了。**」
- 三态话术:收到 = **按了** ｜ 在办 = 留着痕 ｜ 办妥 = **落印**
- 五线延展:① 语音一句话 → 生成一张待按指印的契 ② 进度 = 指纹一圈圈填色 ③ 办妥 = 红指印落下,进"指印册" ④ AI 僚机 = Pinkprint(蓝图) ⑤ 挂坠 = 指纹纹路的金属圆牌,按一下 = 收到。

**占用/商标风险**|**中**
- App Store 中国区搜「指印」:**无同名 App**(返回全不相关)。中文干净。
- 英文 `Pinkprint`:**Nicki Minaj 2014 年专辑《The Pinkprint》**——存在文化占用,音乐类商标可能已注册。软件类(9/42)未见冲突,但需检索。
- 「指印」二字通用度中等,9 类显著性尚可。

## 3. 本疆域 Top 3

### 🥇 Top 1 · **双勾 / Twotick**

**理由:** 唯一一个做到 **「名字 = 符号 = 产品机制」三位一体** 的候选。
- **符号即品牌**:✓✓ 是全球年轻人已被 WhatsApp 训练十年的"已读"符号,零教育成本;同时它又是两只钩住的小指;同时它又是"完成"。**一个符号同时承载了"承诺""收到""办妥"三件事,这在命名里是极罕见的效率。**
- **机制即话术**:勾的个数就是状态,进度条就是"第二个勾正在被画完",挂坠就是一枚 ✓✓ 金属片。五条产品线全部能用同一个视觉语言讲完,不需要发明第二套比喻。
- **可注册性实测干净**:中区无同名(实测返回零相关),英文改用造词 `Twotick` 规避了已被占用的 `DoubleTick`。
- **直击核心痛点**:产品要卖的就是"已读不回"的解药——而 ✓✓ 就是"已读"这个符号本身。**用敌人的符号,打敌人。**
- 代价:它聪明但不煽情,情感浓度低于 Top 2。**建议做法:双勾做主品牌名,"一百年不许变"做 slogan——理性拿名字,感性拿嗓子。**

### 🥈 Top 2 · **不许变 / No Take-Backs**

**理由:** **情感天花板最高 + 可注册性实测最干净(中英双端零结果)。**
- 中文取自童谣里情绪最重的落点,而**英文 `No take-backs!` 是英美小孩拉勾时喊的原句——不是翻译,是同一个仪式的另一种语言。** 一个中文品牌能在英文里找到"原生对应句",可遇不可求。
- 它是这批里**唯一有人格的名字**:嘴硬、较真、有点撒娇。品牌一开口就有声音。
- 整个视觉体系可以只用"印章"一个器物,和硬件挂坠(一枚能真的蘸印泥盖章的坠子)完美闭环——**这是能卖钱的物件。**
- 代价:**它不像一个 App 的名字。** ASO 搜索不友好,用户要"被讲一句话"才懂。这是拿"搜索便利"换"品牌力"的赌注。

### 🥉 Top 3 · **勾妥 / Gotto**

**理由:** **最像一个能活下来的产品名——不炫技,不肉麻,零风险。**
- 纯造词(「勾妥」不是任何现成中文词),显著性强,9/42 类拿证概率高;中区实测无同名。
- 两个字压完全流程:**勾上了(承诺),就得妥(完成)。**
- 英文 `Gotto` 谐音 `gotta`(I gotta do it)——**一个中文造词能在英文里落成"我得去办"这句口语动词,极难得。**
- **它是这批里唯一认真处理了"男性用户不反感"的名字。** 现有 GTM 的粉红 + 小指 + 印章组合,存在把办事方那一侧劝退的真实风险——而这个产品有一半的成败在他愿不愿意打开。「勾妥 / Gotto」+ 靛蓝配色,是这个问题的解。
- 代价:故事性弱,第一眼看不出是情侣产品(但这也许正是优点)。

## 4. 给决策者的三句实话

1. **「拉勾勾」保住概念,放弃名字。** 拉勾网的在先商标 + 招聘品类对 ASO 的绝对占据 + Pinky Promise 在美区的四重撞车——三重压制下,把它做主品牌是在给未来的自己埋雷。**但它的整套视觉语言(小指、印章、说到做到、拉钩上吊一百年不许变)一个字都不用扔,全部可以嫁接到上面任何一个新名字上。**
2. **✓ 是这个产品真正的品牌资产,不是"勾"这个字。** 名字可以换,符号不能换。任何最终选定的名字,都应该能让 ✓ 站在 logo 中心。
3. **本文档所有"已占用"结论均有实测来源**(iTunes Search API 返回的 trackName / sellerName),但 **App Store 检索 ≠ 商标检索**。Top 3 进入决赛前,必须在中国商标网做第 9、35、42 类的正式近似检索——尤其是「三勾」(三勾商城)和「勾妥」(与 Gotta Apps 的相邻类别)这两条。
