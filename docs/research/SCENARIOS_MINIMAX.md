# 场景扩展(minimax 生成,待人工复核)

# 「念响」30 个新使用场景脑暴

> 设计原则:把"说一句"从**表达工具**升级为**关系基础设施**。下面 5 大类,各 6 个,重点押在你们 24 个老用例没覆盖到的——**节日礼数、转折时刻的颗粒度、说不出的话、群体分工的协调、挂坠独占的"手占嘴空"场景**。

---

## 🎊 A. 节日节点(6 个)

### 1. 春节·压岁钱"代发不掉队"
- **一句话情境**: 人在海外,妈问"给七大姑八大姨的压岁钱都发了没",你根本记不全。
- **产品动作**: 妈按住挂坠说"把今年压岁钱清单念一遍",AI 拆成 16 张任务卡,每张标"谁 / 多少 / 祝福语 / 是否已发",孩子端看到就划掉。
- **情绪价值**: 海外游子不焦虑"礼数",长辈不焦虑"孩子忘没忘"。
- **最适配**: 银发版 + 家庭版
- **没想到点**: 把"红包"从一次性转账,变成**有进度可见的家务**。

### 2. 母亲节·"云端陪伴券"
- **一句话情境**: 妈独居,孩子远在北上广,送花送蛋糕她都说"别乱花钱"。
- **产品动作**: 孩子说一句"想陪妈做三件事",AI 生成 3 张"陪伴券"——陪她逛一次超市 / 陪她去一次医院 / 陪她看一次花展。妈可以挑时间"兑换",孩子那边显示"待履约"。
- **情绪价值**: 礼物不束之高阁,陪伴被具体成可执行项。
- **最适配**: 亲子版 + 银发版

### 3. 父亲节·"父子挑战赛"
- **一句话情境**: 爸嘴硬不肯体检,孩子催不动。
- **产品动作**: 孩子用挂坠录"爸,本周末测个血压,办妥了我给你买你想要的茶",AI 生成"父子挑战"任务链,爸每完成一项解锁一泡茶。
- **情绪价值**: 父亲感受到**被尊重的关心**而不是被唠叨。
- **最适配**: 亲子版(反向适用)
- **没想到点**: 把"健康督促"重构成**父子间的游戏契约**。

### 4. 清明·"远程祭扫同步"
- **一句话情境**: 家族散落五地,清明祭扫人手不够,又不能改日期。
- **产品动作**: 家族长按挂坠"今年清明怎么安排",AI 把祭扫拆成 N 段:在沪大伯买花 / 在杭二叔烧纸 / 海外小辈写电子悼词 / 在老家的堂姐打扫祖坟。每个人只看到自己那一段。
- **情绪价值**: 仪式感不被距离稀释,家族感不靠群聊刷屏维持。
- **最适配**: 家庭版 + 群体版

### 5. 冬至·"同一时区不同厨房"
- **一句话情境**: 四姐妹北上广深,冬至都加班,但妈想全家一起包饺子。
- **产品动作**: 妈按住挂坠"冬至咱们一起包",AI 生成同步任务链:剁馅 / 和面 / 擀皮 / 包 / 煮。定时器四地同步启动,妈那边每收到一项"办妥"亮一格,最后一格是"视频一起煮"。
- **情绪价值**: 团圆**跨过屏幕**真的发生。
- **最适配**: 家庭版

### 6. 520 / 七夕·"24 小时情书任务链"
- **一句话情境**: 异地恋,520 不知道咋浪漫,送花又太老套。
- **产品动作**: 一方说"我想你了",AI 不直接转发,生成"24h 情书任务链":早起听一段语音 / 中午收到一张图 / 下班自动开一首歌 / 晚上一个"抱抱"指令。对方每完成一项解锁下一项。
- **情绪价值**: 把"我爱你"三个字拆成**24 小时的细水长流**。
- **最适配**: 情侣版

## 🏠 B. 生活转折(6 个)

### 7. 搬家·"百日作战地图"
- **一句话情境**: 搬家本身一约搬家公司就完事,真正累的是后面 100 件零碎。
- **产品动作**: 一方说"我们要搬家了",AI 生成 3 个月任务链,按周解锁:第 1 周水电网过户 / 第 2 周装窗帘 / 第 3 周改地址……分配到两人,有进度条,每完成一项房子图上"亮一个房间"。
- **情绪价值**: 焦虑被**颗粒化**,争吵减少 80%。
- **最适配**: 情侣版 + 家庭版

### 8. 孕期·"按周解锁任务"
- **一句话情境**: 准妈妈孕吐难受,准爸爸想帮忙但不知道干嘛,问"你需要啥"得到的永远是"不用"。
- **产品动作**: 准妈挂坠"今天第 24 周,反应是 xxx",AI 生成本周任务发给准爸:陪散步 3 次 / 做夜宵 2 次 / 陪产检 1 次。每完成一项,宝宝"长一个能力"(进度条美化)。
- **情绪价值**: 准爸从"焦虑的旁观者"变成"被明确需要的参与者"。
- **最适配**: 情侣版(孕产) + 硬件挂坠
- **没想到点**: 让**不知道怎么做的人**有具体动作可执行,不是给"不知道要什么的人"压力。

### 9. 术后照护·"24 小时轮值表"
- **一句话情境**: 妈做完髋关节手术,兄弟姐妹都想照顾,但时间凑不齐,群里天天"今天谁去"。
- **产品动作**: 老大按住挂坠"妈手术,排个照护表",AI 生成 2 周轮值:谁几点到 / 负责喂药翻身复健 / 谁做饭。每个人收到的卡只是**自己当天那一段**。
- **情绪价值**: 亲情不因照护而磨损,群聊不再刷屏甩责任。
- **最适配**: 家庭版 + 银发版

### 10. 新手铲屎官·"21 天训练营"
- **一句话情境**: 第一次养小狗,乱咬乱叫乱拉,两人对着崩溃。
- **产品动作**: 一方说"我们家狗又拆家了",AI 识别问题,生成"21 天训练营"任务链:今天教坐下 / 今晚定时喂 / 周末打疫苗。两人轮流打卡,进度条长出"小狗技能树"。
- **情绪价值**: "救命它又拉了"→"今天又解锁新技能"。
- **最适配**: 情侣版 + 硬件挂坠

### 11. 宠物临终·"告别清单"
- **一句话情境**: 养了 15 年的狗快走了,全家都难过但不知道怎么告别。
- **产品动作**: 一人说"毛毛可能撑不过这个月",AI 生成"告别清单"任务:本周全家陪它晒太阳 / 拍一组照片 / 写一封信 / 选墓地。完成一项点亮一颗星。
- **情绪价值**: 离别被**温柔地仪式化**,多年后想起不遗憾。
- **最适配**: 家庭版
- **没想到点**: 没有任何市面产品做**临终仪式**这个高情感浓度场景。

### 12. 父母晚年·"不打扰的远程照料"
- **一句话情境**: 父母 60+,孩子在外地,担心他们忘吃药、吃不好、又不愿被"监控"。
- **产品动作**: 孩子预设"爸妈每日节奏",AI 每天给爸妈发小任务卡:早起测血压 / 中午钙片 / 傍晚散步 20 分钟 / 晚上和老姐妹视频。完成情况孩子远端可见但**不催促**。
- **情绪价值**: 父母有"被需要感"不是"被监控感",孩子不焦虑。
- **最适配**: 银发版 + 亲子版

## 🤝 C. 微妙关系(6 个)

### 13. 冷战破冰·"最小台阶"
- **一句话情境**: 情侣冷战中,谁都不想先开口,开口就代表"我输了"。
- **产品动作**: 一人按住挂坠小声说"把牛奶热一下",对方收到不是"和好"压力,而是**一个最小行动**任务,完成后进度条"在办→办妥",自动解锁下一句"今天辛苦啦"。
- **情绪价值**: 给一个**不下台阶也能下**的台阶。
- **最适配**: 情侣版 + 硬件挂坠
- **没想到点**: 把"道歉 / 示弱"这种高风险动作,**降维成低风险小任务**。

### 14. 异地时差·"晨昏接力"
- **一句话情境**: 她在纽约,他在北京,差 13 小时,他醒来她要睡了,日常对话总是错开。
- **产品动作**: 一人发"今天第 x 天",AI 根据双时区生成"晨昏接力链":她下班录段语音 / 他醒来听到 / 他去上班打卡 / 她睡前收到回应。任务按两地时间交错推进。
- **情绪价值**: 时差从"空白"变成**接力赛**。
- **最适配**: 情侣版(异地)

### 15. 不会说出口的道歉·"代办即道歉"
- **一句话情境**: 你意识到错了,但当面说不出口,微信又显得敷衍。
- **产品动作**: 挂坠按住"我想道歉但不会说",AI 把"对不起"翻译成**代办任务卡**:帮你把脏衣服洗了 / 你爱吃的放冰箱 / 明早闹钟帮你定好。对方收到的是"被照顾",不是"对不起"三个字。
- **情绪价值**: **道歉不靠嘴,靠手**。
- **最适配**: 情侣版 + 硬件挂坠

### 16. 与疏远父母破冰·"零压力微任务"
- **一句话情境**: 和爸三年没正经说话,突然联系怕尴尬,直接打电话更不会。
- **产品动作**: 孩子给自己设"和爸说一句",AI 只生成**极小极小**的任务:第 1 周发一张童年照 / 第 2 周问个菜谱 / 第 3 周说"今天吃了爸爱吃的"。每周一项不强迫。
- **情绪价值**: 把"和解"拆成**小到不会失败的连接**。
- **最适配**: 亲子版
- **没想到点**: 多数亲情产品只服务"经常联系的家庭",忽略**3-5 年不说话**的修复期。

### 17. 朋友失恋·"不说话的陪伴"
- **一句话情境**: 闺蜜刚分手,你想关心但不知道说啥,问了又怕变成"审讯"。
- **产品动作**: 你按住挂坠"陪我闺蜜",AI 生成"不打扰陪伴卡":给她点一杯奶茶 / 周末约一次发呆 / **不主动问后续**。每完成一项你那边进度条走一格。
- **情绪价值**: 关心不变成**追问**。
- **最适配**: 闺蜜 / 挚友版

### 18. 暗恋试探·"低风险信号链"
- **一句话情境**: 暗恋一个同事,不确定对方态度,直接表露怕被拒很尴尬。
- **产品动作**: 挂坠"今天靠近一点点",AI 生成"低风险试探链":帮她带杯咖啡 / 借一本书 / 问个小问题。对方回应了系统提示"信号 +1",连续 3 次 +1 解锁"邀约卡"。
- **情绪价值**: 把暗恋的煎熬变成**可量化的小冒险**。
- **最适配**: 单身 / 社交版
- **没想到点**: 给"说不出口"一个**安全退出机制**。

## 👥 D. 群体场景(6 个)

### 19. 室友公约·"隐形管家"
- **一句话情境**: 4 人合租,马桶没人刷 / 垃圾没人倒 / 水电总有人忘。
- **产品动作**: 一人说"把公约立一下",AI 自动生成**轮值表**,每周一轮:谁倒垃圾 / 谁刷马桶 / 谁收快递。完成打勾,未完成挂坠"温柔嘟一下",不伤面子。
- **情绪价值**: 矛盾不靠群聊撕,靠**流程化**。
- **最适配**: 群体 / 合租版

### 20. 闺蜜旅行·"云端行李员"
- **一句话情境**: 三闺蜜约好去日本,分工永远不均,行前吵了三次。
- **产品动作**: 一人挂坠"我们要去 xx 玩 x 天",AI 拆成行前 / 行中 / 行后三大任务包,每个人收到自己的:订机票 / 订民宿 / 做攻略 / 换日元 / 带插头。完成项变绿,谁拖后腿**看得见**。
- **情绪价值**: 旅行前的"谁负责啥"争吵归零。
- **最适配**: 闺蜜 / 旅行版

### 21. 球队备战·"教练分任务"
- **一句话情境**: 业余足球队要打比赛,队长时间难凑齐,微信群通知发完没人应。
- **产品动作**: 队长挂坠"周六比赛安排下",AI 生成本周备战:谁周三加练体能 / 谁带水 / 谁订场地 / 谁接新队员。每个人看到自己那一段。
- **情绪价值**: 球队从"草台班子"变"有组织"。
- **最适配**: 群体版 + 硬件挂坠(运动)

### 22. 拼租账单·"温和催款不尴尬"
- **一句话情境**: 室友水电 AA,总有人拖着不转,直接催又显得计较。
- **产品动作**: 付款方说"这次共 xxx,每人 xx",AI 生成"温和催款卡"发给每位,**未付的人挂坠轻微提示**而不直接通知所有人。完成标记已付。
- **情绪价值**: **钱的事不伤感情**。
- **最适配**: 群体版

### 23. 家委会·"作业陪护轮值"
- **一句话情境**: 班级家委会每周要家长值班陪孩子做活动,总落在几个人头上。
- **产品动作**: 家委按住挂坠"排下周值日",AI 生成周一到周五谁负责:打印卷子 / 送水果 / 校门口迎。每家收到的**只是自己那天的卡**。
- **情绪价值**: 家委会不变成"少数人苦力"。
- **最适配**: 群体版 + 亲子版

### 24. 老年大学·"学习搭子配对"
- **一句话情境**: 退休阿姨上老年大学,想找搭子练书法 / 唱歌,但不知道怎么开口。
- **产品动作**: 一位阿姨挂坠"想找人一起练书法",AI 在同伴中匹配"小老师",生成每日打卡:今天练 xx 字 / 明天去博物馆 / 后天互相点评。挂坠**大字 + 语音播报**。
- **情绪价值**: 退休生活不孤独,学习有同伴。
- **最适配**: 银发版 + 群体版

## 🎙️ E. 硬件挂坠专属(6 个)

> 这 6 个共同特征:**双手被占、视线被占、注意力被占**,但脑子是空的。挂坠=在那些时刻,你的"第二张嘴"。

### 25. 开车·"安全语音待办"
- **一句话情境**: 开车时突然想到给老婆买生日礼物,不能打字也不能解锁。
- **产品动作**: 方向盘旁挂坠按住"记一下,给老婆买……",AI **不弹屏干扰驾驶**,只在停车后推送整理好的"待办任务卡"给她。
- **情绪价值**: 开车不分心,心意不漏掉。
- **最适配**: 硬件挂坠(车载)
- **没想到点**: 多数语音助手是"现在执行",念响可以**延后到安全时执行**。

### 26. 厨房·"手油也能记"
- **一句话情境**: 做饭正切菜,突然想起老公说想吃饺子,但手上全是油。
- **产品动作**: 灶台旁挂坠"嗨,记一下,回家路上买饺子皮",AI 转成"代办卡"推给老公手机,他办妥拍照"已办"。
- **情绪价值**: 做饭的人不被"小事"反复**打断心流**。
- **最适配**: 硬件挂坠(厨房) + 家庭版

### 27. 健身房·"练后打卡同步"
- **一句话情境**: 健身搭子各练各的,练完发"我练了"给对方没动力。
- **产品动作**: 挂坠语音"今天练背,40 分钟",AI 生成"训练卡"同步到对方进度板,两人训练量**周对比**,完成率低于对方时挂坠"懒散小表情"。
- **情绪价值**: 健身搭子从"搭伴"→**搭子 + 对手 + 伙伴**。
- **最适配**: 硬件挂坠(运动)

### 28. 遛狗·"灵感闪念罐"
- **一句话情境**: 遛狗时灵感 / 想到某事 / 想给某人说句话,但回家就忘。
- **产品动作**: 牵引绳上挂坠"嗨,记一下……",AI 捕捉归档,回家时统一整理成"今日闪念清单",再决定发不发。
- **情绪价值**: 不打断散步的心流,也不漏掉灵感。
- **最适配**: 硬件挂坠(户外)

### 29. 抱娃 / 哺乳·"单手发任务"
- **一句话情境**: 新妈妈一手抱娃一手喂奶,不能打字,想让老公回来路上带点东西。
- **产品动作**: 挂坠语音"买 xx",AI 识别后转"代办卡"给老公,他带回来拍照"办妥"。
- **情绪价值**: 新妈妈不被困住,家人有**具体可执行**的支持,而不是"你辛苦了"这种空话。
- **最适配**: 硬件挂坠(育儿) + 亲子版

### 30. 银发极简·"按一下,说我今天有点想你了"
- **一句话情境**: 80 岁奶奶不会用智能手机,想孙女了但打不了字。
- **产品动作**: 挂坠只有**一个超大按钮**,按一下"奶奶想你了",AI 转成语音 + 文字 + 小表情发到孙女手机;孙女点"收到",奶奶那边挂坠**亮一盏灯**。
- **情绪价值**: **不让爱卡在"技术门槛"里**。
- **最适配**: 银发版(极简)
- **没想到点**: 极简本身就是产品语言——一盏灯就是最朴素的情感回报。

## 💡 三条贯穿性洞察(供 PRD 引用)

1. **"代办"是新型表白** — 30 个场景里有 11 个本质上是"用做代替说",这是念响最深的护城河,任何竞品都难抄。
2. **挂坠的本质是"在错位的时空里喊一声"** — 开车 / 做饭 / 哺乳 / 遛狗 / 远在海外,5 个时空错位场景强烈建议**首发挂坠功能 demo 全部覆盖**。
3. **"高情感低频"才是新蓝海** — 临终、破冰、道歉、临终告别,这些场景频率低但情感浓度高、口碑传播力极强,**可作为品牌故事的素材库**而非主营收功能。
