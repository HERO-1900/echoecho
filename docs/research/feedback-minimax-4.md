# MiniMax 用户反馈 #4

> 扮演 3 对【日本情侣】和 3 对【韩国情侣】,用当地视角讲:这个产品在你们国家会不会火、哪个模式水土不服、"女方给男方派任务"在你们文化里会不会有问题(尤其韩国)。

# 双向情侣备忘录 × 6对中日韩情侣的本土化审判

> 测试方法:把产品demo丢给6对真人(描述),让他们用本地方言/思维吵起来。下面是实录整理,**所有判断基于他们正在过的生活,不是产品经理的脑补**。

## 🇯🇵 日本篇

### 1. 田中夫妇 — 33岁&31岁 / 结婚3年 / 东京杉并区 / 共働き

**会不会火?7/10,慢热但能成。**

田中さん(夫)是SIer的项目经理,妻子在博报堂做art。两个人都加班到22点是常态。家里最大的痛点:**电饭煲的米是谁买的、卫生纸什么时候换、洗碗机的滤网谁清理**——这些事不致死,但每次都靠「察する」(察言观色)解决,最后永远是妻子先开口。

> **夫:**「これ、嬉しいけど…正直、リスト化されると“業務報告”っぽくない?」
> **妻:**「業務報告でいいの。察してって言い続けるのがもう無理なのよ。」

他们**真的需要一个"任务卡"系统**,但最大的文化摩擦是:日本夫妻把"嘴上说出来"这件事本身就视为"失败的前兆"。App强行让你**写下"我想要你买牛奶"**,会让很多人觉得"这种话都要说,我们的关系是不是哪里坏了"。

**会火的关键:**默认界面不要叫"任务"(タスク),叫**「おねがいごと」**(请求)或**「暮らしメモ」**(生活备忘)。语音优先+任务化这一招,在日本是绝杀——因为**日本人最讨厌打字,但不讨厌自言自语**。录个"今天回家能顺路去药妆店吗"比打字10倍轻松。

**水土不服的模式:**
- 🔴 **毒舌修勾**:田中夫是典型的**「草食系」+「察し系」**,让他在App里扮演"被骂的小狗"会让他直接卸载。"犬系男子"是约会市场的卖点,不是家里的常态玩法。
- 🟡 **女王陛下**:日本有**「旦那様クエスト」**文化(把老公当支线任务做),所以这个模式**女性用户会买**,但男性用户会在2ch吐槽"婚前昭和婚后令和,我成什么了"。
- 🟢 **念想**:这是日本市场的**主菜**。完美命中「共働き+别居婚+週末婚」人群。

**"女方派任务"在日本有没有问题?**
有,但已经发生了。**「モラハラ妻」**(精神暴力妻)这个词在日本是Google热搜常客。App如果设计上**任务只能单向派发**,会立刻被男性论坛(爆サイ)打成"妻支配ツール"。**必须双向+可见,谁派给谁的任务数要平衡**,日本男的要的不是少做事,是**公平感**。

**硬件挂坠的预判:**这才是日本市场的**真正的杀手锏**。日本夫妻哪怕同住一个屋檐下,也经常是「各个部屋」(各在各的房间)。一个挂坠按一下、对方桌面的小设备亮一下——这种**极简的非语言信号**,简直是为**「以心伝心」文化**量身定做。建议找Sanrio或Miffy联名。

### 2. 佐藤情侣 — 26岁&24岁 / 大阪 / 同棲1年 / 恋爱中

**会不会火?8/10,最容易起量的segment。**

> **佐藤(男):**「正直、彼女の機嫌が読めない時が一番怖い。」
> **彼女:**「読めないんじゃなくて、読もうとしてないでしょ。」

这就是这款App的**理想用户**。20代後半的同棲情侣,正处于"甜蜜期结束、磨合期刚开始"的死亡谷。他们愿意用工具,但讨厌"工具感太强"的东西。

**会火的关键:**五种人格模式切换,日本情侣市场**没有这种东西**。既有的Line+Instagram组合,要么太散,要么太公开(Story被共同好友看到会尴尬)。一个**私人、可切换、不公开**的伴侣专属空间,本身就有市场。

**水土不服的模式:**
- 🟢 **拉勾勾**:这个模式**日本女生会疯掉**。「約束」「記念日」「サプライズ」文化在日本是刚需(圣诞、情人节要回礼、生日要ハナミズキ照片墙...)。但**男生会累**。建议设计成"女生设定玩法,男生只需要点同意"。
- 🟢 **女王陛下+毒舌修勾**:20代后半的关西情侣,**这两个模式会是使用率最高的**。大阪本身就有**「ボケとツッコミ」**文化,角色扮演=本土的漫才。但要小心:**"小狗"模式在大阪男生听起来是"いぬ",不是"かわいい"**,文案需要本土化。
- 🟡 **爸妈听话**:和恋爱关系**完全冲突**,情侣打开这个模式会觉得很怪。可能要走"未来演练"包装,或者直接拆成另一个App叫「家族メモ」。

**"女方派任务"有没有问题?**
**这里日本和韩国有个巨大文化差异**——日本20代的「**JK語**」「**あざと**」文化里,女生"撩"男生、给男生派小任务是**人设的一部分**。所以佐藤情侣**完全没问题**。但日本女生会要求App有**"撒娇语音"模板**——"帮我买布丁回来"录好后一键发,而不是真的打字。这是机会。

**硬件挂坠:**佐藤说"挂坠要能挂在Nintendo Switch的joycon上",这个需求可能不是开玩笑——年轻情侣把伴侣当成"装备"的思路很日本。

### 3. 渡边夫妇 — 56岁&54岁 / 京都 / 结婚28年 / 子供独立

**会不会火?5/10,不是爆款但是长尾,而且最不被抄袭。**

渡边さんは退休的中学老师,太太在区政府做嘱託。两个人最常说的话是"你回来啦""嗯"。

> **渡边さん:**「Line毎日打つのは逆にしんどい。声を聞くだけでいい。」
> **奥さん:**「携帯の字が小さくて見えない。ボタンが大きいのがいい。」

这对的反馈价值连城:

1. **语音优先 + 震动 + 大字模式 + 不需要每天打开**,这四个feature加起来,就是日本**「老後コミュニケーション」**的解法。
2. 挂坠按一下、对方桌面的设备**慢慢亮起来**,渡边さん说"这比LINE好,LINE吵"。
3. 他们完全不懂什么是"AI僚机",但当被解释"你说一句'想吃天ぷら',它直接列出附近3家评分高的",他们**立刻就懂了**——因为这就是他们现在用黄色电话本记东西的方式。

**水土不服的模式:**
- 🔴 **拉勾勾、女王陛下、毒舌修勾**:这三个对渡边夫妇来说是**"看见就想关掉"**的存在。文案太年轻化,模式名都看不懂。要做**"シニア版"**,把念想直接做成**默认皮肤**。
- 🟢 **爸妈听话**:这里要拆。**渡边夫妇没有小孩在身边(女儿在东京)**,他们需要的是**"儿女模式"**,但用法是**"给女儿发一个想你的震动+语音"**,不是"被女儿管"。这个模式在日本的**真正战场是50代和30代之间**,不是情侣。

**会火的真相:**
日本App市场的金科玉律是**「TVCM一打+车站广告一投」**。这款App不投广告没人知道。但一旦投,渡边夫妇这种**沉默的、付费意愿极高、不刷App Store评论**的中老年用户,会贡献**日本市场50%的ARR**。

**给产品经理的警告:**日本老人对**"App内的数据归属"**极度敏感。**必须明确写"录音只保存在你俩的设备里"**,否则会被NHK特别报道打成"监视ツール"。

## 🇰🇷 韩国篇

### 1. 민수(32) & 지은(29) — 首尔 江南区 / 同居2年 / IT行销×品牌经理

**会不会火?6/10,用户会装,但3个月留存率会暴跌。**

韩国市场的**残酷事实**:你做的任何一个沟通类App,都在和**카카오톡(KakaoTalk)**打。KakaoTalk已经不只是聊天工具,是**身份认证、支付、地图、打车**的操作系统。你要情侣**为了"派任务"装第二个App**,用户的心智成本极高。

> **민수:**「카톡으로 하면 안 되나?」
> **지은:**「카톡에 보내면 '응 알았어' 하고 아무것도 안 하잖아. だから終わらないの。」

**지은说到了痛点**——KakaoTalk上沟通"任务"是**没有状态的**,没法追踪。민수看了3秒就回"알겠어",然后忘了。这是这款App在韩国**唯一真实的产品价值**。

**水土不服的模式:**
- 🔴 **毒舌修勾(小狗模式)**:민수听到这个模式直接说**"한국에서 남자가 개 취급 당하는 건 아직... 좀 그래"**。韩国社会是**严格的「호칭 체계」**——오빠/오빠/형/님,称呼就代表权力结构。让30代韩国男人在App里接受"小狗"身份,这不是恶趣味,是**职场尊严危机**。20代或许可以,30代以上**接近于零**。
- 🟡 **女王陛下**:韩国有**공주병(公主病)**这个词,且**舆论对女生公开"女王人设"很苛刻**。지은说"我其实很想要这个模式,但我同事看到会觉得我疯了"——**私密+没人知道是前提**。产品设计必须**完全无外部分享入口**。
- 🟢 **念想**:韩国情侣**最缺的不是浪漫,是生活协作**。지은补一句"洗衣机滤网和요리(料理)采购"是最大战场。念想模式就是它。

**"女方给男方派任务"在韩国有没有问题?——重点来了**

> **민수的原话是:「한국 남자는 시키는 거 싫어하는 게 아니라, 시킨 게 티 나는 거 싫어합니다」**
> (韩国男人不是讨厌被使唤,讨厌的是被使唤这事儿**被别人看见**。)

这是**韩国文化里最精辟的洞察之一**。

韩国社会对男性的评价体系是**"능력"+"체면"**——能力和面子。女朋友在私人聊天里派任务是OK的(甚至有些男生**暗自喜欢**),但:
- **绝对不能被朋友看到**(兄弟群里截图就是社会死亡)
- **绝对不能让女方父母/亲戚看到**(한국 사위(女婿)地位≈乙方)
- **绝对不能在公开平台留痕**(Story、IG posts)

所以这款App在韩国**必须做到**:
1. **没有任何"战绩分享""情侣任务打卡"等功能**——韩国男生会直接拒用。
2. **"派任务"在UI上要叫"부탁"(拜托),不叫"지시"(指示)**。一个词的差别,决定留存。
3. **任务完成率**对韩国男生来说**是负担**——和军队里"점호"(点名)文化撞车。要弱化。

**硬件挂坠:**
민수说"이거 군대에 있는 사람한테 미친 거 같은데"——**这才是真正的韩国市场**。韩国男性**2年内必须有1-2年去军队服役**,期间的沟通工具是**군인 위문 App(군인앱)**。挂坠+震动+语音=把**"我想你"做成硬件**,击中每年20万入伍的男生的痛点,也击中留守女方的痛点。

### 2. 현우(23) & 수아(22) — 首尔↔釜山 / 异地恋 / 현우明年3月入伍

**会不会火?9/10,这个人群是天然的early adopter。**

수아在釜山读研,현우在首尔IT公司上班,异地一年半,**明年현우入伍后这App就是他们的命**。

> **수아:**「현우가 군대 가면 카톡도 잘 못 봐요. 面会도 2주에 한 번. 振動だけでいいから感じたい。」
> **현우:**「部隊에서 폰 检查 들어가는데 記録が残らない语音だけ聞いてくれる機能があれば。」

**这两个需求太真实了**:
- 군대(军队)期间**手机会被定期检查**,所以**纯语音消息(不存留)+即时发送+24h后消失**这个功能,需要立刻做。
- 面会(면회)时不让用手机,但允许带**佩戴物**——硬件挂坠**有面会场景的天然通路**。

**水土不服的模式:**
- 🟢 **拉勾勾**:이 커플의 메인 모드. 韩国年轻情侣的**「기념일 중독」**(纪念日上瘾)和**「카톡 프로필 매일 바꾸기」**(头像每天换)文化,App要能自动生成"明天是100天纪念日,任务:送花"。
- 🔴 **爸妈听话**:현우家里爸爸是**가부장(家长制)**传统,수아和未来公婆关系=韩国婚恋的**第一杀手**。这个模式**绝对不能出现在情侣App里**,会和"효도(孝道)"文化撞车产生奇怪的伦理感。
- 🟡 **毒舌修勾**:在20代韩国男性中,**"강아지상"**(小狗长相/性格)是**2023-2025的当红人设**。현우说"나 강아지상이라"的时候是笑着的。所以这个模式在20代**男生会主动开**,但对**수아来说会很得意**——这是个**罕见的韩日性别错位**。在日本是女生玩小狗,韩国是男生主动当小狗。

**"女方派任务"在韩国年轻人:有可能但有副作用。**

수아说了一段很诚实的话:
> 「솔직히 남자친구가 何か頼んでくれたら嬉しいけど、내가 계속 시키면 그 사람 친구들한테 '그녀가 공주예요'라고 욕먹어요」

韩国**没有"小公主"是褒义词的语境**。20代女生让男朋友跑腿,会被女生朋友说"대단해",但**会被男生那边的朋友圈打成"공주병"**。App要解决这个问题,得做**"双方任务量周报"**——让数据证明**女方也为男方派了任务**,心理对冲。

**硬件挂坠的战略级机会:**
현우说了一句**价值1000亿韩元的话**:
> 「入隊 전 2개월에 사귀는 사람 많은데、이거 서로 주면 입대 선물로 완벽한데」

韩国**入伍前的两个月是恋爱高峰期**(很多情侣在入伍前分手/订婚/感情升温)。如果挂坠定价在3-5万韩元(¥150-250),定位成**입대 선물/졸업 선물**,再加上便利店/쿠팡合作铺货,有望**直接吃到韩国男性消费市场的红利**。

### 3. 정호(35) & 예린(33) — 釜山↔首尔 / 恋爱4年 / 明年婚礼 / 同居准备中

**会不会火?5/10,人群存在但付费意愿低。**

정호是釜山造船所的高级工程师,예린在首尔做财务,**正在筹备婚礼和婚礼后的신혼집(新婚房)**。这个阶段的核心痛点是:**装修、采购、家电选择、婚礼guest list**。

> **예린:**「결혼준비で一番ケンカになるのは、결혼식 예산과 신혼여행 行き先。카톡으로 할 수록 싸워요。」
> **정호:**「남자는 화장실 가서 한숨 쉬고, 여자는 인스타에 스토리 올려요。」

**AI僚机模式**对他们来说是**救命功能**。"想买洗衣机"→三家对比+预算建议+促销信息,这等于**直接把결혼준비 컨설턴트(婚庆顾问)塞进App**。韩国婚庆市场的中间服务费动辄200-500万韩元,一个能干的AI僚机值这个价的1/10。

**水土不服的模式:**
- 🟢 **念想**:完全契合"同居准备"的阶段。
- 🔴🔴 **爸妈听话**:这是韩国市场的**最大地雷**。韩国情侣/夫妻关系里**「시모(婆家) vs 신부(新娘)」**是**终极压力测试**。一个App里如果有任何"父母模式"和"情侣模式"混在一起的UI,会让예린**立刻警觉**:"这是不是要让我未来婆婆看我们的聊天?"**物理隔离这两个模式**,或者**干脆不做爸妈模式,把它单独做成Family版**。
- 🔴 **女王陛下**:정호是**35岁的韩国男人,釜山,造船所,10年资历**。他的朋友圈子对"女朋友是女王"这件事**零容忍**。这个模式在30代+韩国男性身上**几乎完全失败**。
- 🟡 **毒舌修勾**:정호反应是"나는 강아지 아니에요, 고양이"(我不是小狗,是猫)。韩国30代男性的**강아지상(小狗相)趋势已经退潮**,现在是**고양이상(猫相)**。**产品模式名字要改**,改"毒舌修勾"为"毒舌猫"或"호구(傻狗)"**会有差异化**。

**"女方派任务"在韩国30代+婚姻预备期:更加敏感。**

예린的原话:
> 「결혼하면 시집살이 시작되는 게 한국이에요. 남편한테 task 주는 건 좋은데, 姑母가 '우리 며느리는 남편 부려먹는다' 이러면 끝이에요」

(结婚后在韩国就是婆家生活的开始。让老公做任务本身没问题,但**大姑说"我们儿媳把儿子当奴才"就完了**。)

**韩国的"派任务"问题不是男女关系问题,是家族政治问题。**

这款App要进韩国市场,**必须**:
1. **默认开启"친족 차단"(亲族屏蔽)**——聊天记录、任务列表、模式设置,全部不能有任何形式被导出/截图/分享。
2. **绝对不做"战绩海报""情侣成就"等任何可分享UI**。
3. **进入韩国市场前,找一位韩国女性marketing+一位传统婚俗顾问做cultural review**,否则会在**네이트판/블라인드(职场论坛)**被打成"중국발 가부장적 앱"。

**硬件挂坠:**
对**정호&예린这种异地+准备同居的情侣**,挂坠的场景是**"装修时两个人分头跑家具城,按一下互相call back"**。这个场景太具体了,产品宣传片可以拍一条:**江南家具城,예린在看沙发,按挂坠,정호在另一家店的试衣镜边,口袋里的设备震了一下**。这条片子在韩国的转化率会极高。

## 最后的判断 (硬核总结)

| 维度 | 🇯🇵 日本 | 🇰🇷 韩国 |
|---|---|---|
| **核心杀手锏** | 硬件挂坠(以心伝心文化)+ 语音优先(讨厌打字) | 군대 场景 + 입대 선물 营销 |
| **最容易爆的模式** | 念想 + 拉勾勾 | 念想 + 拉勾勾 |
| **最大地雷模式** | 毒舌修勾(草食系不接受) | 女王陛下(30代)+ 爸妈听话(触碰시모红线) |
| **"派任务"的文化风险** | モラハラ妻标签 → 必须双向可视 | 공주병 + 婆家文化 → 必须亲族屏蔽+零分享 |
| **竞品/生态** | LINE的整合压力 | KakaoTalk的操作系统级压力 |
| **建议首发城市** | 东京/大阪(20-30代同棲人群) | 首尔/釜山(20代异地+입대人群) |
| **不要做的事** | 不要叫"タスク",叫"おねがいごと" | 不要做"成就/分享",做"封闭私域" |
| **必做的事** | 和三丽鸥联名 | 找군대 멘탈 케어 App 联名/被集成 |

**一个反向洞察送给产品经理:**
这款App在**日本是中老年人的真需求,在韩国是20代年轻人的真需求**——这两个市场打法完全不同,几乎可以当两个产品做。
而在中国(产品母市场)其实**最有想象空间**——年轻女生的"派任务"文化已经被验证,男生侧的"小狗人设"正在被娱乐化(参见"忠犬八公"梗),且中国没有KakaoTalk级别的整合者,**微信小程序+独立App双轨**是可行路径。

最后一句:
> **日本男人要的是"被公平对待",韩国男人要的是"被看得起",中国男人要的是"被理解"。**
> 同一款App,三个市场三种卖法。祝你好运。
