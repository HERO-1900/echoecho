## 特调开关(demo 层)

- **E 适老开关**:body 字号 ×1.25,统计卡收起为一行,任务卡只显示 done 大按钮+语音键放大;进度条隐藏百分比数字改文字"在办呢"
- **C 安全阀**:切入 C 时弹一次确认("这是双人玩法模式,对方也看得到你的用语风格。随时可一键切回。")——对应品牌页"规则先于游戏"
- **D 通缉令**:超 48h 未动的 received 卡片加"已鸽 N 天"角标(玩梗,不发对方通知)
- 模式仅改本机呈现,不改数据;对方看到的是自己模式下的话术(设备级人格)
