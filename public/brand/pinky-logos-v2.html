Brand Proposal v2 · 推倒重来

拉勾勾
是两只小指,不是两个铁钩。

v1 被全盘否决的根因不是"画得不好",是画错了东西——它由 A17 17 0 1 1 这类纯圆弧拼成两个互相咬合的金属钩,没有拳头、没有指关节、没有拇指,大脑只能读成五金件。 v2 的第一动作不是画,是去看一张真实的照片(Wikimedia Commons《Pinky swear closeup》),从人手解剖重建几何: 握拳的体块 + 朝内的指关节起伏 + 竖起的拇指 + 从下方升起的小臂。拳头承载"这是手",勾住只是中间那个很小的细节。10 个变体共用同一套底层骨架,只换"画法/媒介"。

为什么 v1 翻车 / v2 修了什么

v1 的三个致命错误

1. 没有手,只有钩。纯圆弧构成,没有拳头体块,大脑没有"手"的落脚点,只好读成钩子 / 锚 / 问号。

2. 等宽工业线条 + 完美圆弧。真手指有关节、会变粗细、指尖是圆润指腹。完美圆弧正是机械感的来源。

3. 抽象度过高。"拉勾勾"是童年、是撒娇、是体温,需要肉感;过度几何化把温度全蒸发了。

v2 的四条修正(全 10 版共用)

1. 拳头是主角。每只手 = 圆润拳体 + 3 个朝内的指关节 + 竖起的拇指(最强的"这是手"信号,v1 完全没有) + 从画面下方升起的小臂。

2. 手是镜像,结是旋转。照片给的答案:两手左右镜像,但两根小指绕结心 180° 旋转对称,才可能真正互穿——此前"纯镜像"推演必然把两指糊成一坨,根因在此。

3. 小指分掌节与勾节两段(粗→细),指尖带圆润指腹,缠绕是有机螺旋而非正圆。

4. 一高一低 = 两个人。右手整体下移、略小,不靠性别符号、靠身形差异说"这是两个人"。
